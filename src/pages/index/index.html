Technologies

Programming languages and technologies used for developing this app are listed below:

Frontend: Cordova
Frontend UI: Ionic Framework
Backend Server: ExpressJS
Database: MongoDB - MLab
Image Upload: Cloudinary
Backend Deploy: Heroku
Realtime: Socket.io
Authentication: jsonwebtoken
Validation: Joi
